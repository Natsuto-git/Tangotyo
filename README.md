# Tangotyo - 単語帳アプリ

Webブラウザで動作する単語学習アプリです。ObsidianのMarkdown表形式で作成した単語データを読み込んで学習できます。

## 機能

- **Markdown表形式のデータ読み込み**: Obsidianなどで作成した単語データを簡単に読み込み可能
- **デッキ切り替え**: 異なる言語や科目ごとにデータを分けて保存・管理
- **学習モード**: 単語→意味、意味→単語の両方向学習に対応
- **進捗管理**: 正解/不正解の記録と統計表示
- **ブックマーク機能**: 重要な単語をマークして後で確認
- **シャッフル学習**: ランダム順での学習
- **ミス単語再学習**: 不正解だった単語だけをもう一度学習
- **単語一覧**: 全単語の確認とミス単語のみの表示

## 使い方

1. ブラウザで `index.html` を開く
2. テキストエリアにMarkdown表形式の単語データを入力（例）:
   ```
   | 単語 | 意味 | 例文 |
   |---|---|---|
   | hello | こんにちは | Hello, nice to meet you. |
   | goodbye | さようなら | Goodbye, see you later. |
   ```
3. 「Load」ボタンをクリックしてデータを読み込み
4. 「Save」ボタンでデータをブラウザに保存（任意）
5. 学習を開始！

## URLパラメータ

- `?k=english` : 英語デッキを指定
- `?k=chinese` : 中国語デッキを指定
- `?k=french` : フランス語デッキを指定

## データ保存

- 入力データはブラウザのlocalStorageに保存されます
- デッキごとに別々に保存されるため、複数の言語を同時に学習可能
- ブックマークや学習履歴も保存されます

## 対応ブラウザ

- Chrome, Firefox, Safari, Edgeなどのモダンブラウザ
- スマートフォン・タブレット対応
